# ADP 프로토타입 v2

제칠일안식일예수재림교회 교인 교육 플랫폼 프로토타입 — [ADP_기능명세.md](./ADP_기능명세.md) 기반.

3개 면이 하나의 서비스를 구성합니다.

| 면 | 진입 | 대상 |
|---|---|---|
| ADP 앱 (모바일) | `/adp` | 교인 · 목회자 |
| 바이블가이드 앱 (모바일) | `/guide` | 바이블가이드 |
| 관리자 페이지 (PC) | `/admin` | 교회/합회/연합회 관리자 |

- `/` 런처에서 세 앱을 오가며 체험, `/preview` 에서 전 화면 바로가기
- 데이터는 시드 + localStorage (런처의 "데이터 초기화"로 리셋)
- 디자인: 모던 프리미엄 — 로고 딥블루(#1B4F8A) 기반 쿨 뉴트럴 토큰, Pretendard

## 실행

```bash
npm install
npm run dev
```

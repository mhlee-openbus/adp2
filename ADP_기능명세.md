# ADP 기능 명세

> 제칠일안식일예수재림교회 교인 교육 플랫폼 — **현재 구현된 기능 목록** (기능 관점만 기술)
> 3개 면(ADP 앱 · 바이블가이드 앱 · 관리자 페이지)이 하나의 서비스를 구성한다.

---

## 1. 공통 기능

### 1.1 계정 · 역할
- 계정 타입 2종: **목회자 / 일반회원**
- 계정 위에 얹히는 역할:
  - **관리자 권한** — 레벨 3단계(교회 / 합회 / 연합회). 상위 레벨일수록 넓은 범위 열람
  - **바이블가이드** — 회원 중 지정. 관심자 전도 관리 담당
- **교인** = 교육에 등록되어 교육 단계를 가진 회원
- **관심자** = 아직 교인이 아닌 사람. 회원 계정이 아닌 별도 데이터(바이블가이드가 관리)

### 1.2 두 개의 5단계 체계
| 단계 | 교육 단계 (교인) | 관심자 단계 |
|---|---|---|
| 1 | 영혼구원 | 준비 |
| 2 | 정착 | 심기 |
| 3 | 교육 | 양육 |
| 4 | 지도자양성 | 추수 |
| 5 | 세계선교 | 보존 |

### 1.3 권한 범위
- 교회 관리자 = 자기 교회 / 합회 = 산하 교회들 / 연합회 = 전체
- 모든 목록·통계·질문은 로그인한 사용자의 권한 범위로 자동 필터링

### 1.4 런처
- 서비스 진입 인덱스: ADP 앱 / 바이블가이드 앱 / 관리자 페이지 선택
- 전 화면 미리보기(`/preview`) · 데이터 초기화
- 각 로그인 화면에서 "← 다른 앱 보기"로 런처 복귀 가능

## 2. ADP 앱 (모바일) — 교인 · 목회자

### 2.1 로그인 / 회원가입
- 모드 선택 로그인: **교인** / **목회자·관리자**
- 회원가입: 이름 + 계정 타입(일반회원/목회자) 선택
  - 일반회원 가입 → 즉시 교인 등록, 시작 단계 1.영혼구원 자동
  - 목회자 가입 → 승인 방식 추후 확정(안내만)

### 2.2 교인 모드 — 하단 탭: 대시보드 · 강의 · 질문

**대시보드**
- 현재 단계 표시 + 5단계 여정 시각화(완료 → 현재 → 예정)
- 현재 단계 진도율(%)
- **이어보기**: 다음에 볼 강의 바로 진입
- 현재 단계 강의만 노출(지난/다음 단계 비노출)

**강의 목록**
- 현재 단계 패키지의 강의를 순서대로 표시
- **순차 잠금**: 앞 강의를 완료해야 다음 강의 해제(잠긴 강의는 🔒 비활성)
- 강의별 완료 여부 · 영상 출처(유튜브/비메오/자체 업로드) 표시
- 강의 미등록 단계는 "준비 중" 안내

**강의 시청**
- 영상 영역 + 강의 제목 + 다음 강의 안내
- **"완료" 버튼**으로 시청 완료 처리 → 다음 강의 잠금 해제
- 단계의 마지막 강의 완료 시 **자동 승급** → 승급 완료 화면

**승급 완료**
- "○단계 완료! 다음은 △단계" 축하 + 새 단계 여정 표시
- "새 단계 강의 보기"로 이어서 학습

**질문 (1:1 게시판)**
- 질문 작성(제목 + 내용) → 본인 질문 목록에 쌓임
- 질문별 **답변 대기 / 답변 완료** 상태 표시
- 질문 상세: 질문 + 답변 스레드 열람, **추가 문의** 작성 가능
- 본인이 올린 질문만 열람 가능(1:1)

### 2.3 목회자 모드 — 하단 탭: 우리교회 · 교인 · 질문

**우리교회 (현황 대시보드, 보기 전용)**
- 요약 지표 4종: 전체 교인 / 이번 달 신규 / 평균 진도율 / 관심자 수
- **교인 단계 분포**(1~5단계 인원 막대) — 막대 클릭 시 해당 단계로 필터된 교인 목록으로 이동
- **최근 등록 교인** 5명(이름·단계·등록일, 클릭 시 상세)
- **전도 현황**: 관심자 / 미배정 / 교인 전환 수
- 권한 레벨에 따라 보는 범위 자동 확장(교회 < 합회 < 연합회)

**교인 목록**
- 권한 범위 내 교인 전체 목록(이름 · 단계 칩 · 현재 단계 진도바)
- **이름 검색** + **단계 필터 칩**(전체/1~5) — 우리교회 분포에서 넘어오면 자동 적용
- 교인 클릭 → 상세

**교인 상세 (보기 전용)**
- 개인정보: 연락처 · 이메일 · 생년월일(만 나이) · 성별 · 주소 · 교육 등록일
- 5단계 교육 여정 시각화
- 현재 단계 진도율(%) + **단계별 진도**(1~5단계 각각 완료/전체 강의)

**질문 (1:다 답변)**
- 권한 범위 내 교인 질문 전체 목록 + 필터(전체/대기/완료) + 대기 건수
- 질문 펼침 → 본문·스레드 확인 → **답변 등록**
- 교인의 추가 문의에 재답변 가능

**모드 전환**
- 상단바에서 **교인 보기 ⇄ 목회자 보기** 전환(로그아웃 불필요)
- 전환 대상에 실제 데이터가 있을 때만 버튼 노출

## 3. 바이블가이드 앱 (모바일) — 전도 도구

하단 탭: **오늘 · 관심자 · 미션**

### 3.1 로그인
- 바이블가이드 계정 선택 로그인(담당 관심자 수 표시)

### 3.2 오늘 (메인 대시보드)
- **내 미션**: 바이블가이드 본인의 오늘 활동 1건(관심자 미션과 구분된 카드) — 완료 체크/취소
- **관심자 미션 목록**: 오늘 미션이 있는 담당 관심자 카드(이름 · 단계 · 할 활동 · 완료 표시)
- 카드 클릭 → 관심자 상세
- 빈 상태: 관심자 0명 → 등록 유도 / 오늘 미션 없음 → 안내

### 3.3 관심자
- **단계 분포 요약**(준비~보존 인원 막대)
- 담당 관심자 전체 목록(최근 등록순, 이름 · 단계 · 교인 전환 뱃지)
- **관심자 등록**: 이름(필수) · 전화번호 · 관계(복수 태그: 가족/친구/동료/미분류 + 자유 입력) · 기념일(복수: 종류+날짜) — 시작 단계 1.준비 자동
- 본인이 담당하는 관심자만 열람

### 3.4 관심자 상세 (핵심 화면)
- **오늘의 미션** 최상단(알고리즘 추천 표기) — 완료 체크/취소
- 이름 + 현재 단계 + 교인 전환 여부
- 5단계 관심자 여정 시각화 + **승급/강등**(양방향 수동)
- **교인으로 전환** (전환 후 상태 유지 표시)
- **1:1 화상 시작** → 화상 화면(양쪽 비디오 · 함께 보는 영상 · 채팅 3분할 레이아웃)
- 더보기(접힘): 전화·관계·기념일·비공개 메모 열람/수정

### 3.5 미션 (히스토리)
- 담당 관심자의 **모든 미션을 미션 단위로** 최신순 나열
- 필터: **전체 / 예정 / 완료** (건수 표시)
- 각 항목: 날짜(오늘 뱃지) · 완료/예정 상태 · 활동 내용 · 대상 관심자(이름+단계)
- 항목 클릭 → 해당 관심자 상세

## 4. 관리자 페이지 (PC) — 운영

좌측 메뉴: 대시보드 · 강의 큐레이션 보드 · 교인 관리 · 관심자 현황 · 바이블가이드 관리 · 질문 관리 · 권한/조직 관리

### 4.1 로그인
- 관리자 계정 선택 로그인(권한 레벨 표시)
- 권한 없는 회원 접근 차단

### 4.2 대시보드 (통계)
- 핵심 3지표: **교인 단계별 분포** · **관심자 단계별 분포** · **신규 유입 추이**(최근 6개월)
- 분포 카드 클릭 → 해당 관리 화면 이동
- 권한 레벨에 따라 집계 범위 자동 변경

### 4.3 강의 큐레이션 보드
- 5단계 섹션 세로 스택, **교회별 패키지** 구성(상위 관리자는 교회 선택)
- 강의 **추가**(제목 + 출처: 유튜브/비메오/자체 업로드 + **영상 URL**)
- 강의 **편집 · 삭제 · 순서 변경**(↑↓)
- URL이 있는 강의는 링크 표시(새 탭)
- 강의 없는 단계 → 교인 앱에서 "준비 중"

### 4.4 교인 관리
- 권한 범위 내 교인 목록(이름 · 교회 · 단계 · 진도 · 가입 경로) + 이름 검색
- **수동 등록**: 이름 + 교회 + 시작 단계 지정(기본 1)
- **단계 직접 조정**(수동 승급/강등) — 목록에서 바로 변경

### 4.5 관심자 현황
- 권한 범위 내 관심자 목록(이름 · 단계 · 담당 바이블가이드 · 교인 전환 여부)
- 이름 검색 + 단계 필터
- **담당 바이블가이드 지정/재지정**(미배정 포함) — 담당이 바뀌어도 진행 기록(미션·메모·단계)은 유지
- 관리자가 직접 **관심자 추가**(교회·담당 지정 가능)
- 상세 보기(전화·관계·기념일·메모)

### 4.6 바이블가이드 관리
- 바이블가이드 목록(이름 · 교회 · 담당 관심자 수)
- **역할 부여**: 기존 회원 검색 → 지정
- **역할 해제**: 담당 관심자는 전원 **미배정**으로 풀림 → 관심자 현황에서 재지정

### 4.7 질문 관리
- 권한 범위 내 교인 질문 전체(제목 · 작성자 · 교회 · 날짜 · 답변 상태)
- 검색(제목·작성자) + 필터(전체/답변 대기/답변 완료) + 대기 건수
- 질문 펼침 → 본문·스레드 확인 → **답변 등록**(1:다)

### 4.8 권한 / 조직 관리
- 조직 트리 열람(연합회 > 합회 > 교회 — 보기 전용)
- 회원별 **관리자 권한 부여/회수 + 레벨 지정**(교회/합회/연합회 단일 축)

## 5. 관통 규칙 (동작 원칙)

1. **교육 승급**: 현재 단계 모든 강의 완료 → 자동 승급 (5단계에서 정지). 관리자는 수동 조정 가능
2. **순차 잠금**: 앞 강의 완료 없이는 다음 강의 시청 불가
3. **관심자 단계**: 바이블가이드가 수동 승급/강등(양방향). 교인 전환도 바이블가이드가 결정
4. **투웨이 매칭**: 바이블가이드가 등록하면 자동 자기 담당 / 관리자가 재지정 가능. **관심자가 데이터의 주인** — 담당 교체 시 기록 유지
5. **질문 게시판**: 교인은 1:1(본인 질문만) / 목회자·관리자는 1:다(범위 내 전체 답변)
6. **권한 범위**: 모든 화면이 로그인 사용자의 레벨(교회⊂합회⊂연합회)에 맞춰 자동 필터
7. AI 관련 표현은 **"알고리즘"**으로 표기
